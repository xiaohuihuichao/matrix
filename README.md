# matrix_with_ann
This C++ code include a matrix library and a multi-layer perceptron(full connection network) with computer graph.

# Compile:
```
	$cd matrix
	$mkdir build
	$cd build
	$cmake ..
	$make
```

# Run:
```
	$./00matrix_Main
	Or
	$./module/Ann/02Ann
```
